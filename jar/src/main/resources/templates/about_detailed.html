REACT

사용자 인터페이스를 만들기 위한 선언적이고 효율적이며 유연한 JavaScript 라이브러리입니다. '컴포넌트'라는 작고 고립된 코드 조각을 사용하여 복잡한 UI를 구축할 수 있습니다.

핵심 컨셉: Virtual DOM

React는 실제 DOM의 가상 표현을 메모리에 유지하여, 상태 변경 시 이전 가상 DOM과 비교해 변경된 부분만 실제 DOM에 적용합니다. 이 '조정(reconciliation)' 과정은 DOM 조작을 최소화하여 애플리케이션의 성능을 크게 향상시킵니다.

주요 특징

UI를 재사용 가능한 독립적인 부분으로 나누어 개발 생산성과 유지보수성을 높입니다.
데이터가 부모에서 자식으로 단방향으로 흐르므로, 데이터 흐름을 예측하고 디버깅하기 쉽습니다.
함수형 컴포넌트에서 상태(State)와 생명주기(Lifecycle) 기능을 사용하여 더 간결하고 직관적인 코드를 작성할 수 있습니다.

JAVASCRIPT

웹 페이지에 생동감을 불어넣는 동적인 스크립팅 언어입니다. 현대 웹 개발의 핵심 기술로, 프론트엔드와 백엔드(Node.js) 모두에서 사용됩니다.

핵심 컨셉: 비동기 처리

JavaScript는 단일 스레드로 동작하지만, 이벤트 루프와 콜백, Promise, async/await를 통해 비동기 작업을 효율적으로 처리합니다. 이를 통해 네트워크 요청이나 대용량 파일 처리 중에도 UI가 멈추지 않고 사용자에게 쾌적한 경험을 제공할 수 있습니다.

주요 특징

클래스 기반 언어와 달리 프로토타입을 통해 객체지향 상속을 구현합니다.
함수를 변수에 할당하거나 다른 함수의 인자로 전달하는 등 객체처럼 다룰 수 있어 함수형 프로그래밍이 가능합니다.
변수의 타입이 런타임에 결정되어 유연한 코드 작성이 가능합니다.

TYPESCRIPT

JavaScript에 정적 타입 시스템을 추가한 상위 집합(Superset) 언어입니다. 대규모 애플리케이션에서 코드의 안정성과 유지보수성을 크게 향상시킵니다.

핵심 컨셉: 정적 타이핑

개발 단계에서 변수나 함수의 타입을 명시하여, 코드가 실행되기 전 컴파일 단계에서 타입 관련 오류를 미리 발견할 수 있습니다. 이는 런타임 에러를 줄이고, 코드의 의도를 명확하게 만들어 협업을 용이하게 합니다.

주요 특징

타입을 명시하지 않아도 컴파일러가 코드를 분석하여 자동으로 타입을 추론해 개발 편의성을 높입니다.
복잡한 객체 구조를 정의하고, 재사용 가능한 유연한 컴포넌트를 작성할 수 있습니다.
최신 ECMAScript 표준을 지원하며, 이를 구형 브라우저에서도 동작하는 코드로 컴파일할 수 있습니다.

FIGMA

웹 기반의 실시간 협업 UI/UX 디자인 도구입니다. 아이디어 구상부터 프로토타이핑, 개발자 핸드오프까지 디자인 프로세스 전반을 지원합니다.

핵심 컨셉: 실시간 협업

여러 명의 디자이너, 기획자, 개발자가 하나의 파일에 동시에 접속하여 작업하고 피드백을 주고받을 수 있습니다. 이를 통해 커뮤니케이션 비용을 줄이고 디자인 작업의 효율성을 극대화합니다.

주요 특징

재사용 가능한 UI 요소와 스타일을 시스템으로 관리하여 디자인의 일관성을 유지합니다.
콘텐츠 변화에 따라 자동으로 레이아웃을 조정하여 반응형 디자인을 쉽게 구현할 수 있습니다.
별도의 툴 없이 인터랙티브한 프로토타입을 제작하여 사용자 흐름을 테스트하고 아이디어를 검증할 수 있습니다.
