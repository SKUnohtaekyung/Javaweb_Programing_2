SKILLS & TOOLS

생각의 도구들

REACT

컴포넌트 기반의 재사용 가능한 UI를 구축하고, 가상 DOM을 통해 효율적인 렌더링을 구현합니다. 상태 관리 라이브러리(Redux, MobX)와 함께 사용하여 복잡한 애플리케이션의 데이터 흐름을 관리한 경험이 있습니다.

JAVASCRIPT

ES6+ 문법에 익숙하며, 비동기 처리를 위한 async/await와 Promise를 능숙하게 사용합니다. 웹 API를 활용하여 동적인 기능을 구현하고, 다양한 라이브러리와 프레임워크의 기반이 되는 핵심 개념을 깊이 이해하고 있습니다.

TYPESCRIPT

정적 타이핑을 통해 코드의 안정성과 가독성을 높이는 개발 방식을 선호합니다. 인터페이스, 제네릭, 유틸리티 타입을 활용하여 복잡한 데이터 구조를 명확하게 정의하고, 컴파일 시점에서 오류를 사전에 방지합니다.

FIGMA

아이디어를 빠르게 시각화하고, 인터랙티브 프로토타입을 제작하여 사용자 경험을 검증합니다. 개발자와의 원활한 협업을 위해 디자인 시스템과 컴포넌트 기반의 디자인을 설계하고, 일관성 있는 UI를 제공합니다.
